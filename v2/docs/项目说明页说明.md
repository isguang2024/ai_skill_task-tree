# 项目说明页说明

前端导航已新增“文档”页，用于集中展示 Task Tree V2 的用途、入口、能力分类和使用方法。

## 页面内容

- 项目用处
- 访问入口
- 如何使用
- 能力分类
- Memory 规则
- Run 的用途
- 升级与排查

## 适用人群

- 第一次接触项目的人
- 需要快速回忆能力边界的人
- 需要确认 V1 / V2 区别的人
- 需要确认 Run / Memory 语义的人

## 对应前端入口

- 文档页路径：`/docs`
- 导航菜单名称：`文档`

## 说明

这是一页面向使用者的说明页，不替代 `skill/`、`docs/` 或 `backend/docs/`，而是把常见信息聚合到一个容易点开的入口里。
